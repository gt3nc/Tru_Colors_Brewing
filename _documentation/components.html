Centralized

Centralized will center your blocks vertically and horizontally.

This is going to be centralized content.

<div class="centralized">
    <h3>This is going to be centralized content.</h3>
</div>

Usage in application.js

$('.centralized').centralized({
    delay: 1500,
    fadeSpeed: 500
});

Heads up! Usage requires that object above has position: relative;

JavaScript /assets/js/plugins/jquery.centralized.js is requred.

Fixed On Later

Show header navigation

This plugin will display navigation when user's scroll down one page height.

You will need to initialize this plugin.

$('#navigation').fixedonlater({
    speedDown: 250,
    speedUp: 100
});

Heads up! JavaScript /assets/js/plugins/jquery.fixedonlater.js is requred.
