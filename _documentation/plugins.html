Bx slider

Treble uses BxSlider for first page page-welcome H1 taglines.

This plugin can be used as:

  • Fully responsive - will adapt to any device
  • Horizontal, vertical, and fade modes
  • Slides can contain images, video, or HTML content
  • Advanced touch / swipe support built-in
  • Uses CSS transitions for slide animation (native hardware acceleration!)
  • Full callback API and public methods
  • Small file size, fully themed, simple to implement
  • Browser support: Firefox, Chrome, Safari, iOS, Android, IE7+
  • Tons of configuration options

Heads up! JavaScript /assets/js/plugins/jquery.bxslider.min.js is requred.

More information can be found at http://bxslider.com/ and github

HashLoader

HashLoader will give you power to load pages using ajax requests.

Using hashLoader uses code>data attributes to findout which URL should be loaded and where to add content. You can also define action how to load and what to do after loading is finished.

<a href="#pages/portfolio/project_1.html" data-destination="portfolio-items" data-insert="before" data-action="hide">Load my page</a>
Data Description
href URL of page you want to load. Relatve path, but could also be absolute.
data-destination

data-destination will be container where you want to load page. In combination with data-insert this doesnt have to be empty div.
data-insert

data-insert will be the way you want to load new content. There are several ways to do this:

  • append - This is default action, it will append content indside data-destination.
  • before - Will add content before data-destination.
  • after - Will add content after data-destination.
  • prependTo - Will add content inside of data-destination but at the top.
  • appendTo - Will add content inside of data-destination but at the bottom.
  • html - Will replace content of data-destination.
data-action

data-action will give you option to remove or hide link for loading ajax page. There are several ways to do this:

  • remove - This is default action, it will append content indside data-destination.
  • hide - Will add content before data-destination.

Heads up! JavaScript /assets/js/plugins/jquery.hashloader.js is requred.

One Page Nav

A lightweight jQuery plugin for the navigation on one-page sites. Uses the ScrollTo Plugin plugin to add smooth scrolling when clicking on the navigation. Also adds a class to the correct navigation items as you are scrolling through the different sections.

Treble uses this plugin for #main-menu navigation.

    $('#main-menu').onePageNav({
        currentClass: 'active',
        changeHash: false,
        scrollOffset: navigationHeight - 10,
        scrollThreshold: 0.5,
        scrollSpeed: 750,
        filter: '',
        easing: 'swing',
     });

Heads up! JavaScript /assets/js/plugins/jquery.nav.min.js is requred.

More information can be found at github

Parallax

Parallax is, parallax, it uses CSS background and create parallax effect on user scroll.

Treble uses this plugin for #page-welcome, #page-features and #page-twitter.

        $('#page-welcome').parallax("0%", 0.2);
        $('#page-features').parallax("0%", 0.07);
        $('#page-twitter').parallax("0%", 0.1);

Just add CSS bacground and adapt JS configuration to your needs.

Heads up! JavaScript /assets/js/plugins/jquery.parallax-1.1.3.js is requred.

Treble has disabled parallax for devices which have width size less then 980px.

        if(pageWidth > 980){
            $('#page-welcome').parallax("0%", 0.2);
            $('#page-features').parallax("0%", 0.07);
            $('#page-twitter').parallax("0%", 0.1);
        }

This configuration can be found at /assets/js/plugins/application.js.

More information can be found at this location.

Responsive Videos

Tweet

Treble uses Tweet to display latest Tweets. This plugins uses JS but since Twitter changed API from 2013-06-11, this plugin is requiring also PHP.

This plugin is modified by Stan Scates for https://github.com/StanScates/Tweet.js-Mod.

This is how it works from JS side:

    $('.twitterfeed').tweet({
        modpath: 'plugins/twitter/',
        username: 'TheGridelicious',
        count: 3
    });

Configuration for Tweeter AIP is in \plugins\twitter\index.php, where you will need to enter:

  • Your Twitter App Consumer Key
  • Your Twitter App Consumer Secret
  • Your Twitter App Access Token
  • Your Twitter App Access Token Secret

As stated in "How to use" for this plugins, yous should first get your Twitter App access.

Heads up! JavaScript /assets/js/plugins/jquery.tweet.min.js is requred.

More information can be found at this location.

Tweet Carousel

This pulgin will help you to setup Carousel from Twitter bootstrap in combination with Tweet to display latest Tweets.

Plugin will create HTML markup based on numbers of Tweet's visible.

This is how it works from JS side:

    $('#twitterfeed-slider').tweetCarousel({
        interval: 7000,
        pause: "hover"
    });

Heads up! JavaScript /assets/js/plugins/jquery.tweetCarousel.min.js is requred.
