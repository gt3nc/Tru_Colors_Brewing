Icons

Social icons

Social icons are comming in severeal variants. Class social-icons on warapper elements will define href as social icon item.

There are 5 possible social icons which can be used alone or in combination.

  • They are easy extendable by editing /assets/subjective/icon-set.less
  • And adding images to /assets/images/css/icon_set.png

Hover transition on elements

About page

We have created special hover transition for page About. This hover transition is used only on about page, but it can be easy added to other pages.

Style code can be found in file /assets/subjective/page-about.less. In this example, hover transition is used with social icons.

Hover is activated on .item. On hover event block .about-overlay will with the help of CSS3 transition, show it sefe.

  • Facebook Twitter LinkedIn Youtube Pinterest 
<ul class="thumbnails about-items">
    <li class="span4 center">
        <div class="item"> 
            <img class="img-circle" src="http://placehold.it/260x260" alt="Treble">
            <div class="about-overlay img-circle">
                <div class="social-icons sicon-white"> 
                    <a href="#" class="sicon-facebook"><i>Facebook</i></a> 
                    <a href="#" class="sicon-twitter"><i>Twitter</i></a>
                    <a href="#" class="sicon-linkedin"><i>LinkedIn</i></a>
                    <a href="#" class="sicon-youtube"><i>Youtube</i></a>
                    <a href="#" class="sicon-pinterest"><i>Pinterest</i></a>
                </div>
            </div>
        </div>
    </li>
</ul>

Heads up! We disabled CSS3 transition for mobile devices by overwriting it in file /assets/style/objective/responsive-767px-max.less. This is compromise we had to accept since olde version of Android browsers have problem with transitions (they dont have it implemented).

Bug that was showing when using transition was in position: fixed elements, such as navigation which was disappearing.

Style code can be found in file /assets/subjective/page-work.less. In this example, hover transition is used name of project.

Hover is activated on li. On hover event block .portfolio-overlay will with the help of CSS3 transition, show it sefe.

Overlay

Overlay will create effect of network. This option can be used for example in case when you have full width page, and background image.

If the background image is blured or it's to big to be put in full resolution (it's expected to be 2000px in width or more), then you could use this option to mask image.

    <div class="overlay">
    </div>
</div>

Submenu

Adding submenu is simple, just add class submenu to ul element.

Submenu is fluid, it will be in center, and you can add as much as you need elements. I will also have good look on all responsive resolutions.

  • Branding

Divider

Divider after element

To add line divider you should add class line-divider to for example h4.

Divider is added using CSS after element, and it's expecting to have margin-bottom for element you are adding divider.

Heading text


<h4 class="line-divider center">Heading text</h4>

Heads up! It's expected that element will be center aligned.

Divider inline with element

To add line divider you should add class line-divider-inline to for example h4.

Divider is added using CSS after element.

Heading text


<h4 class="line-divider-inline center">Heading text</h4>

Heads up! It's expected that element will be center aligned.

There is restriction in number of characters, but this could be changed by editing image which is used. /assets/images/css/divider_more_text.png
